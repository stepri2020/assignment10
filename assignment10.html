Covid Cases in the US

The API is chose is called COVID-19-API. It has up-to-date information about COVID-19 cases around the world, broken up by country. I found it on the GitHub list of public API's (https://github.com/public-apis/public-apis). I used an XMLHttpRequest to get the data from the API. This API would be helpful for travel websites looking to advise what places have higher or lower COVID cases, or for a website with a map tracking hotspots in the US.
